Owner Onboarding

Visit /admin/onboarding for a guided setup wizard. Progress is saved so you can close the page and return later.

  1. Brand & Logo – enter your outlet name and logo URL.
  2. Tables Count – specify how many tables to create.
  3. Menu Import – upload a CSV or spreadsheet of menu items.
  4. Printer Test – send a test print to verify receipts.
  5. Payment Mode – configure UPI, cash or card options.
  6. Alerts Rules – set up notification rules for events.

The wizard tracks the last completed step and resumes from where you left off using the new /api/onboarding/<id> endpoint.
